# Maltego-Based OSINT Analysis Project

This repository contains the HTML site created to visualize and present my academic project on **OSINT (Open Source Intelligence) analysis using Maltego**. The site serves as a companion to the written report and video presentation, offering an accessible way to explore the workflow, findings, and validation process.

---

## Project Overview
- **Objective:** To design a reproducible OSINT workflow using Maltego CE as the primary tool.  
- **Focus:** Mapping networks, uncovering relationships, and validating findings against credible sources.  
- **Deliverables:**  
  - Written project report (with appendix screenshots).  
  - HTML site for interactive visualization.  
  - Video presentation (4:55 minutes).  

---

## Tools & Technologies
- **Maltego CE** – Core tool for network mapping and visualization.  
- **SpiderFoot** – Secondary tool for automated OSINT scanning.  
- **Wikidata** – Data sources for entity information.  
- **Congress.gov / CourtListener / Freedom of the Press Foundation etc.** – Verification sources.  
- **HTML, CSS, JavaScript** – Used to build the presentation site.  

---

## Site Contents
The HTML site includes:
- **Project Introduction** – Context and goals.  
- **Maltego Overview** – What Maltego is and a summary of its operational logic.  
- **Target Profile** – Overview of the chosen target and why they were chosen.  
- **Results** – Output content and visualization.  
  - **Visual Reconstruction** – Tree list view for easy analysis and interaction.  
  - **Maltego Graphs** – Screenshots of raw Maltego network mapping.   
  - **Verification/Evidence** – Trusted sources used for cross-checking.
