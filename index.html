<PROJECT_ABSTRACT />


This project focuses on learning how to gather information using Maltego, a tool commonly used in cybersecurity for research and investigation. The main goal is to collect and organize publicly available information about a selected person to build a detailed profile of their online presence.


The project utilizes Maltego's features to find digital footprints, identify social media accounts, and map connections between different pieces of information found online. All information gathered comes from public sources that anyone can access.
